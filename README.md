# 🔑 Glassmorphic Password Generator

A lightweight, secure, and visually stunning web application that generates strong random passwords instantly. Built entirely using pure HTML, modern CSS (Glassmorphism), and Vanilla JavaScript.

![Project Preview](https://placeholder.com) *(Optional: Replace this link with a screenshot of your app once live)*

## ✨ Features

*   **Instant Generation:** Generates an 8-character secure alphanumeric password with one click.
*   **One-Click Copy:** Copies the generated password to your clipboard smoothly via the Clipboard API.
*   **Glassmorphism UI:** Features a modern, translucent design interface with animated background gradients.
*   **Fully Responsive:** Optimized flawlessly for desktops, tablets, and mobile screens.
